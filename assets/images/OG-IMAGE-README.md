# Open Graph Image Requirements

## Missing File
You need to create: `og-image.png` in this directory

## Specifications
- **Dimensions:** 1200 x 630 pixels (required for Facebook/LinkedIn)
- **File format:** PNG or JPG
- **File size:** Under 1MB
- **Aspect ratio:** 1.91:1

## Content Recommendations
The image should include:
- PaymentHood logo
- Tagline: "Enterprise Payment Orchestration Platform"
- Clean, professional design with brand colors
- Readable text even when scaled down

## Tools for Creation
- Canva (has Open Graph templates)
- Figma
- Adobe Photoshop
- Or use online OG image generators

## Current Configuration
The site is configured to use `/assets/images/og-image.png` as the default social media preview image for all pages.

Once created, this image will appear when:
- Pages are shared on Facebook
- Pages are shared on LinkedIn
- Pages are shared on Twitter
- Links are previewed in messaging apps
