Installation Guide

Follow these simple steps to install PaymentHood in your Phoca Cart store

1

Download the Plugin

Download the latest PaymentHood Phoca Cart plugin from our GitHub repository. It is a standard Joomla extension package (.zip) — no build steps or extra dependencies required.

Download Plugin
Tip: Always download the latest version for the newest features and security updates.
2

Navigate to the Joomla Extensions Manager

Log in to your Joomla administrator panel and go to Extensions → Install to open the extension installer.

3

Upload & Install the Plugin

Choose the Upload Package File tab, select the ZIP file you downloaded, and click Upload & Install. Joomla will install the PaymentHood plugin in seconds.

4

Enable PaymentHood in Phoca Cart

Once installed, go to your Phoca Cart configuration and navigate to Payment Methods. Find PaymentHood in the list and enable it.

5

Complete Setup of PaymentHood

Open the PaymentHood payment method settings. Click the Complete Setup button to be redirected to PaymentHood for account configuration.

Quick Setup: Sign up with just one click using your Gmail account. No complex forms or lengthy registration process.
6

Setup Your Gateways in PaymentHood Console

You will be redirected to the PaymentHood Console where you can add your payment gateways for Sandbox or Live environments. Once done, you will automatically return to your Phoca Cart store.

Our Strong Recommendation: Set up your Sandbox app and test everything before going Live. Use each environment button to complete the setup in the PaymentHood Console.
All Set! Your Sandbox environment is ready for testing. Once satisfied, follow the same steps to activate your Live environment and start accepting real payments.
