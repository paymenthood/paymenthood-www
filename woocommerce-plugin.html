Installation Guide

Follow these simple steps to install PaymentHood in your WooCommerce store

1

Download the Plugin

Download the latest PaymentHood WooCommerce plugin from our GitHub repository.

Download Plugin
Tip: Always download the latest version for the newest features and security updates.
2

Navigate to WordPress Plugins

In your WordPress dashboard, go to Plugins → Add New Plugin to begin the installation.

3

Upload & Install

Click Upload Plugin at the top of the page, select the ZIP file you downloaded, then click Install Now.

4

Activate the Plugin

Once installation completes, click Activate Plugin to enable PaymentHood in your WooCommerce store.

5

Complete Setup of PaymentHood

After activation, PaymentHood appears in your plugins list. Click the Complete Setup link to begin gateway configuration.

The setup screen will open. Click the Setup Complete button to proceed to PaymentHood.

Sign Up with Gmail

After activation, you'll be redirected to PaymentHood for easy sign-up using your Gmail account. Once completed, you'll automatically return to WooCommerce.

Quick Setup: Sign up with just one click using your Gmail account. No complex forms or long registration process.
6

Setup Your Gateways in PaymentHood Console

You'll be redirected to the PaymentHood Console where you can add your gateways for Sandbox or Live account.

Our Strong Recommendation: Setup your Sandbox app and test everything before going Live. Use each environment button to complete the setup in PaymentHood Console.
7

Enable PaymentHood at Checkout

After setting up Sandbox it will appear as activated. Select "Enable PaymentHood at checkout" to test Sandbox for your shop.

All Set! Your Sandbox environment is ready for testing. Once satisfied, follow the same steps to activate your Live environment and start accepting real payments.
