How It Works

Up and Running in 5 Minutes

Three simple steps from download to accepting live payments — the standard Joomla way.

1
Download the Plugin

Grab the latest release from GitHub. It's a standard Joomla extension package (.zip) — no build tools or extra steps required.

2
Install via Extension Manager

In your Joomla admin go to Extensions → Install, upload the zip and install. Then enable the PaymentHood plugin in Phoca Cart's payment settings.

3
Connect Your PaymentHood Account

Enter your PaymentHood API key in the plugin configuration, select sandbox or live mode, save — and your Joomla store is ready to accept payments.
