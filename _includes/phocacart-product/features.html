Features

Everything Your Joomla Store Needs

One plugin connects your Phoca Cart store to a world of payment options — installed and configured the Joomla way.

🔌

Native Phoca Cart Plugin

Registered as a standard Phoca Cart payment plugin — installs via the Joomla Extension Manager with no manual file copying.

🌍

15+ Global Providers

Connect PayPal, Stripe, Authorize.net, Adyen, Braintree, and many more — all through a single PaymentHood account, configured outside Joomla.

⚡

Automatic Order Management

Phoca Cart order statuses update automatically on payment success or failure. PDF invoices and order notifications fire as normal.

🔒

Sandbox & Live Modes

Test your complete checkout flow in sandbox mode before going live. Toggle environments instantly from the Joomla plugin settings.

📦

Install in Three Clicks

Upload the .zip through the Joomla Extension Manager, enable the payment plugin in Phoca Cart settings, paste your API key — done.

🎛️

Centralized Provider Dashboard

Manage all provider credentials, transactions, and configuration from your PaymentHood account — keeping your Joomla admin clean.
