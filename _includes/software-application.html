{% comment %} SoftwareApplication JSON-LD for an integration product page. Pass the plugin's data object as the `app` parameter, sourced from _data/integrations/.yml (the `software` key on that file). Required fields on `app`: name, operating_system, download_url. Optional: version, release_notes. Constant fields (category, free-price offer, author) are baked in here; `description` comes from the page front matter and `installUrl` is derived from page.url (assumes the product page lives at /integrations//). {% endcomment %}
