Features

Everything Your J2Commerce Store Needs

One extension connects your store to a world of payment options — with full native J2Commerce compatibility.

🔌

Native J2Commerce Integration

Registered as a standard J2Commerce payment plugin — works seamlessly inside the J2Commerce checkout on Joomla 4 and 5.

🌍

15+ Global Providers

Connect PayPal, Stripe, Authorize.net, Adyen, Braintree, and many more — all through a single PaymentHood account.

⚡

Automatic Order Management

J2Commerce order statuses update automatically on payment success or failure. No manual order handling required.

🔒

Sandbox & Live Modes

Test your full checkout flow in sandbox mode before going live. Switch environments with a single toggle in the plugin settings.

📦

Install & Enable — That's It

Install the package from the Joomla Extension Manager, enable the payment plugin, paste your API key, and you're ready to accept payments.

🎛️

Centralized Provider Dashboard

Manage provider credentials, view transactions, and configure settings from your PaymentHood account — not inside Joomla.
