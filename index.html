Get Started in Minutes

Simple, powerful API designed for developers. Process payments with just a few lines of code.

RESTful API with OpenAPI 3.0 documentation
REST API with comprehensive documentation
Sandbox environment with test data
Full API Docs Try Sandbox
JavaScript 
// Process a payment with PaymentHood
const paymenthood = require('@paymenthood/sdk');

const client = new paymenthood.Client({
  apiKey: 'ph_live_...',
  environment: 'production'
});

// Create a payment intent
const payment = await client.payments.create({
  amount: 2999, // $29.99
  currency: 'USD',
  customer: 'cust_abc123',
  description: 'Premium Plan Subscription',
  routing: {
    strategy: 'intelligent', // Auto-route to best gateway
    fallback: true
  }
});

console.log(`Payment created: ${payment.id}`);
