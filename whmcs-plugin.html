Visual Installation Guide

Follow these simple steps with visual guidance to install PaymentHood in your WHMCS:

1

Download & Upload Plugin Files

Download the PaymentHood plugin and upload the files to your WHMCS installation directory.

Extract the downloaded ZIP file and upload all contents to your WHMCS root directory
2

Access Apps & Integration

Navigate to the Apps & Integration section in WHMCS.

Navigate to: Apps & Integration to view available integrations
Apps & Integration list showing available services
3

Select PaymentHood Gateway

Select PaymentHood from the available payment gateways.

Select PaymentHood from the available gateways
4

Gateway Activated Successfully

After pressing activate, the PaymentHood gateway is automatically activated. No action needed - continue to add the addon.

PaymentHood gateway successfully activated
This happens automatically - no action required. Continue to the next step.
5

Select PaymentHood Add-on

Access the PaymentHood add-on from the Apps & Integration section.

Navigate to: Apps & Integration to access PaymentHood add-on
Find PaymentHood in the Apps & Integration section
6

Configure PaymentHood Settings

Configure your PaymentHood integration settings including API credentials and preferences.

Click "Save Changes" to save the configuration
Don't forget to click the "Save" button after entering your configuration settings.
7

Activate & Authorize PaymentHood

Complete the activation and authorization process to connect your WHMCS with PaymentHood.

Navigate to: System Settings → Add-on Modules to access the activation section
Click "Activate"
→
Redirect to PaymentHood
→
Grant Authorization
→
Return to WHMCS
After pressing activate, you'll be redirected to PaymentHood to grant authorization. Once authorized, you'll return automatically to WHMCS.
8

Installation Complete!

PaymentHood is now successfully activated and ready to process payments in your WHMCS.

PaymentHood is now active and ready to use
Your WHMCS is now connected to PaymentHood! Define your bank account in the PaymentHood system and start processing payments.
