Installation Guide

Follow these simple steps to install and configure the PaymentHood plugin for WHMCS.

1

Download & Upload

Download the plugin package and upload it to your WHMCS modules directory.

/path/to/whmcs/modules/gateways/paymenthood/
  • Extract the downloaded ZIP file
  • Upload the "paymenthood" folder to /modules/gateways/
  • Ensure proper file permissions (755 for directories, 644 for files)
2

Activate Gateway

Enable PaymentHood in your WHMCS admin panel under Payment Gateways.

  1. Login to WHMCS Admin Area
  2. Navigate to Setup → Payment Gateways
  3. Find "PaymentHood" in the list
  4. Click "Activate"
3

Get API Credentials

Obtain your PaymentHood API keys from the merchant dashboard.

  1. Login to PaymentHood Dashboard
  2. Navigate to Settings → API Keys
  3. Copy your Public Key and Secret Key
  4. Note your Merchant ID
4

Configure Settings

Enter your API credentials and configure gateway settings in WHMCS.

Setting Description Required
Merchant ID Your PaymentHood merchant identifier Yes
Public Key Public API key for client-side operations Yes
Secret Key Private API key for server-side operations Yes
Test Mode Enable for testing with sandbox environment No
Gateway Priority Primary gateway selection (Auto-routing recommended) No
5

Test Integration

Verify the integration with test transactions before going live.

  1. Enable Test Mode in gateway settings
  2. Create a test invoice in WHMCS
  3. Process payment using test card: 4111 1111 1111 1111
  4. Verify payment status in WHMCS and PaymentHood dashboard
  5. Disable test mode when ready for production

Troubleshooting

Common Issues

Gateway not appearing in payment options

  • Verify plugin files are uploaded to correct directory
  • Check file permissions (755 for directories)
  • Ensure gateway is activated in WHMCS admin
  • Clear WHMCS cache and refresh admin panel

Payments failing or not processing

  • Verify API credentials are correct
  • Check test mode settings match your environment
  • Ensure webhook URL is correctly configured
  • Check PaymentHood dashboard for error logs

Webhook notifications not received

  • Verify webhook URL in PaymentHood dashboard
  • Check server firewall allows incoming connections
  • Ensure webhook.php file has proper permissions
  • Test webhook URL manually
