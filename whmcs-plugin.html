Installation Guide

Follow these simple steps to install PaymentHood in your WHMCS

1

Download the Module

Download the latest PaymentHood WHMCS module from GitHub releases.

Tip: Always download the latest version for the newest features and security updates.
2

Extract & Upload Files

Extract the downloaded ZIP file and upload the contents to your WHMCS installation directory.

# Upload structure:
/your-whmcs-root/
├── modules/
│ ├── gateways/
│ │ └── paymenthood.php
│ └── addons/
│ └── paymenthood/
└── ...
Important: Make sure you upload to the correct WHMCS root directory, not a subdirectory.
3

Activate Payment Gateway

Navigate to System Settings → Payment Gateways in your WHMCS admin panel.

  1. Visit Apps & Integration
  2. In the search box, search for PaymentHood
  3. Select PaymentHood from the results
4

Activate PaymentHood Gateway

Click the "Activate" button to enable PaymentHood in your WHMCS.

5

Sign Up with Gmail

After activation, you'll be redirected to PaymentHood for easy sign-up using your Gmail account. Once completed, you'll automatically return to WHMCS.

Quick Setup: Sign up with just one click using your Gmail account. No complex forms or long registration process.
6

Gateway Ready to Use

Your PaymentHood gateway is now activated and configured. Both sandbox and live apps have been automatically created and are ready to use.

All Set! Your sandbox and live apps have been created automatically. Simply define your gateway settings and start accepting payments.
